# 具身智能全栈学习计划

记录具身智能 (Embodied AI) 全栈学习路线、代码分析、论文笔记与实验记录的个人知识库。

## 目标

- 系统化掌握具身智能从感知、决策、规划到控制的全链路技术
- 深入剖析主流开源项目源码（VLA / Diffusion Policy / RL / Sim2Real 等）
- 沉淀论文阅读笔记与复现实验，形成可复用的知识资产

## 目录结构

| 目录 | 内容 |
| --- | --- |
| [01-基础理论](01-基础理论/) | 机器人学、深度学习、强化学习、控制论等基础 |
| [02-感知模块](02-感知模块/) | 视觉、点云、多模态感知 |
| [03-决策与规划](03-决策与规划/) | 任务规划、运动规划、LLM Planner |
| [04-控制与执行](04-控制与执行/) | 关节控制、阻抗控制、MPC |
| [05-端到端模型](05-端到端模型/) | VLA、扩散策略、强化学习策略 |
| [06-仿真平台](06-仿真平台/) | Isaac Sim / MuJoCo / Gazebo / Genesis 等 |
| [07-代码分析](07-代码分析/) | 主流开源项目源码逐行解析 |
| [08-论文笔记](08-论文笔记/) | 论文精读与对比 |
| [09-实验记录](09-实验记录/) | 复现实验、训练日志、问题排查 |

## 学习路线（迭代中）

1. **基础**：机器人学 → 深度学习 → 强化学习 → 模仿学习
2. **感知**：2D/3D 视觉 → 多视角融合 → 视触觉
3. **策略**：BC → Diffusion Policy → VLA → RL Fine-tune
4. **系统**：仿真训练 → Sim2Real → 真机部署
