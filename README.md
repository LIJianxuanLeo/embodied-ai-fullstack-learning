## 使用建议

- 每章的 `README.md` 是知识点大纲与学习清单
- 章节内可以自由建子目录：`论文/`、`代码分析/`、`笔记/`、`实验/`、`assets/`
- 重要图示统一放在仓库根 `assets/` 或章节内 `assets/`

## 学习节奏

按章节顺序最稳，但允许"分形展开"——先把每章过一遍 0.1 版，再回头深挖每个子主题。
