# 第 7 章 · 机器人的出生与成长：从技术到产品

> 算法 demo ≠ 产品。本章从工程化、商业化、伦理三个角度看：怎么把一个能跑的 policy 变成一个能卖、能持续运营的机器人产品。

## 7.1 硬件平台与本体
- 机械臂：UR、Franka、xArm、Kinova、国产（睿尔曼 / 大族 / JAKA）
- 移动底盘：Spot、AGV、AMR
- 双臂 / 协作机器人：ALOHA、Mobile ALOHA、AgileX
- 人形：Tesla Optimus、Figure、1X、Unitree H1/G1、Fourier、宇树、Booster、智元、星动、银河通用…
- 末端：二指夹爪、吸盘、灵巧手（Inspire / PsiBot / Shadow）
- 选型考虑：负载、精度、价格、生态、可维护性

## 7.2 系统工程
- 计算栈：边缘（Jetson Orin / x86 mini PC）、云端、模型蒸馏/量化（INT8 / FP8 / TensorRT）
- 实时性：高频控制 + 低频感知/策略的解耦
- 通信：EtherCAT、CAN、ROS2 DDS
- 安全：急停、力矩限位、碰撞检测、Functional Safety（ISO 13849 / IEC 61508）
- 可观测性：日志、回放、远程监控

## 7.3 数据飞轮
- 部署 → 失败案例自动捕获 → 人工标注 / 自动纠错 → 数据池 → 再训练
- 在线评估指标设计
- 用户授权与隐私合规

## 7.4 商业模式
- **toB 工业**：物流、汽车、3C、新能源
- **toB 商业**：餐饮、清洁、配送、巡检、医疗辅助
- **toC**：陪伴、家务、教育（最难）
- RaaS（Robot-as-a-Service） vs 直接销售
- 单位经济模型（Unit Economics）：硬件成本 / 部署成本 / 人工节省

## 7.5 生态与玩家
- 海外：Boston Dynamics、Figure、1X、Physical Intelligence、Skild AI、Covariant
- 国内大厂：华为、字节、美团、京东、小米
- 国内创业：智元、宇树、星动纪元、银河通用、千寻智能、它石智航、加速进化、Booster Robotics 等（持续变化，自行更新）
- 资本与产业链：减速器、伺服、芯片、训练数据公司

## 7.6 伦理、安全与法规
- 失效模式与责任归属
- 人机协作安全标准（ISO 10218、ISO/TS 15066）
- 数据合规：GDPR / 个保法
- 就业、社会接受度

## 7.7 自我定位
- 我擅长 / 想做什么：算法？系统集成？数据？硬件？产品？
- 选公司还是创业？大厂还是创业公司？工业还是消费？
- 中长期假设：3 年内哪个细分会跑通商业闭环？
